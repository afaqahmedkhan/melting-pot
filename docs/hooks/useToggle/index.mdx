# useToggle

```
const { on, toggle } = useToggle(); 
```

## Examples

<Playground>
  {() => {
    let { on, toggle } = useToggle();
    
    return (
      <div>
        <button onClick={toggle}>
            {on ? 'on' : 'off'}
         </button>
      </div>
    )
  }}
</Playground>

## API

useToggle() returns an Object

* on: [boolean] Status if on or off
* toggle [function] Has function that toggles 'on' state.
